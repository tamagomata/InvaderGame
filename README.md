## 遊び方

### 操作方法
- **移動**: 矢印キーの左右で操作
- **攻撃**: スペースキーで弾を発射

### ゲームルール
- **制限時間**: なし
- **勝利条件**: すべてのエイリアンを倒すとゲームクリア
- **敗北条件**: 敵の赤い弾が当たるとゲームオーバー
- **得点システム**: エイリアン1体倒すごとに10点獲得

### ポイント
- 戦略的にエイリアンを倒しながら、敵の攻撃を避けることが重要です
- すべてのエイリアンを倒すとゲームクリアとなります
